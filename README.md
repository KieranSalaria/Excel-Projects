# Excel-Projects

## Project Overview

This repository contains a collection of Excel projects completed during the first week of my bootcamp course. The focus of this week was to build a strong foundation in Microsoft Excel, developing essential skills for data handling, analysis, and presentation.

The projects demonstrate my ability to work with structured datasets, apply core Excel functions, and transform raw data into clear, meaningful insights. Each file reflects practical exercises designed to mirror real-world data tasks and improve accuracy, efficiency, and confidence when working with spreadsheets.

## Skills & Tools Used

* Microsoft Excel
* Data cleaning and organisation
* Spreadsheet formatting and layout design
* Logical thinking and problem-solving
* Attention to detail and data accuracy

## Topics Covered

* Basic Excel navigation and worksheet management
* Data entry best practices
* Cell referencing (relative, absolute, and mixed)
* Common formulas and functions (e.g. SUM, AVERAGE, COUNT, IF)
* Sorting and filtering data
* Basic data validation
* Creating and formatting tables
* Introductory charts and visualisation
* Improving readability and usability of spreadsheets

## Purpose of This Repository

The purpose of this repository is to:

* Track my progress throughout the bootcamp
* Showcase foundational Excel skills to employers and collaborators
* Build a reference point for future data and analytics projects
